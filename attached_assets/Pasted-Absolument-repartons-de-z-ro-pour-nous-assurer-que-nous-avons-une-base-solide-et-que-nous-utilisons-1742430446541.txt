Approche recommandée : Utilisation de Flexbox avec une structure HTML simplifiée

Pour ce type de mise en page, Flexbox est l'outil le plus adapté. Il offre une flexibilité et une fiabilité exceptionnelles pour l'alignement des éléments.

2. CSS avec Flexbox

Voici le code CSS correspondant :

CSS

body {
    margin: 0;
    font-family: sans-serif;
    height: 100vh;
}

.container {
    display: flex;
    height: 100%;
}

.panel {
    flex: 1; /* Les panneaux occupent une part égale de l'espace */
    display: flex;
    flex-direction: column;
    justify-content: center;
    align-items: center;
}

.left {
    background-color: black;
    color: orange;
}

.right {
    background-color: orange;
    color: black;
}

/* Styles supplémentaires pour le contenu */
.name-title, .large-name {
    display: flex;
    gap: 10px;
}

.first {
    color: blue;
}

/* Ajoutez vos autres styles ici */
Explications :

body { height: 100vh; } : Garantit que le corps de la page occupe toute la hauteur de la fenêtre.
.container { display: flex; height: 100%; } : Crée un conteneur Flexbox qui occupe toute la hauteur de la page.
.panel { flex: 1; } : Permet aux panneaux de se développer pour occuper une part égale de l'espace disponible.
.left et .right : Définissent les couleurs de fond et de texte pour chaque panneau.
Styles supplémentaires : Ajoutez vos styles spécifiques pour le contenu des panneaux.
